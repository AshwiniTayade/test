    CSS Animation is a techniqe used to change the appearance and behavior of various elements in wed peges. It will control the elements by changinng their motions or display. It has replaced the Animation created by flash and javascript. the animation is created using the @keyframe rule.

    It has two parts CSS properties (describe the animation of the elements) keyframes(specific time intervals at which the animation have to occur) when the animation is created in the @keyframerule,it must have a selector otherwise , the animation will have no effect.

    @keyframe keyframes are the foundation of CSS Animation It will control the intermediate steps in a CSS animation sequence It defines the display animation at the corresponding stages in the Whole duration
